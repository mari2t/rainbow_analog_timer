## Overview

アナログタイマーです。  
1～60分まで1分刻みで時間を設定できます。  
残時間がわかりやすいように、1/4ごとに色を分けました。

Analog timer.  
Time can be set from 1 to 60 minutes in 1-minute increments.  
Colors are separated for each quarter to make it easy to see the remaining time.

## Features

- Next.js
- Tailwind CSS
- Shadcn
